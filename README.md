# windows-edr-lite
 A lightweight endpoint detection response application for personal use.

 As of right now it lists processes, PID, and parent PID. It also has a detection engine with some rules in place already such as malicious activity being detected with a winword.exe parent process and powershell.exe. It will be fine-tuned with more alerts and a user interface as I continue developing it. Below is an example screenshot from my own console.
